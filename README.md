# Java_Study
Java의 정석을 바탕으로 Java에 대한 기본 교육 및 간단한 실습을 진행하는 저장소입니다.
---
### 학습 요약
#### 2026-06-28
- **학습 내용**: Java 기초 - 변수와 입출력 (`_ch02`)
- **요약**:
    - **변수의 선언과 초기화**: 데이터를 저장하기 위한 메모리 공간 확보 및 기본값 설정 방법 학습.
    - **화면 입력 (Scanner)**: `Scanner` 클래스를 사용하여 사용자로부터 문자열과 숫자를 입력받는 실습.
    - **변수의 활용**: `VarEx1`, `VarEx2` 예제를 통해 변수의 값 교환 및 출력 포맷팅 이해.
#### 2026-06-29
- **학습 내용**: Java 기초 - 연산자 (`_ch03`)
- **요약**:
    - **증감 연산자**: 전위 증감(`++i`)과 후위 증감(`i++`)의 연산 우선순위 및 값 할당 방식 학습.
    - **형변환 (Casting)**: 데이터 타입 간의 형변환(Casting) 원리와 정수/실수를 문자형(`char`)으로 변환하는 실습.
    - **산술 연산과 오버플로우**: 정수형 연산 시 오버플로우 방지를 위한 형변환 및 `Math.round()`를 이용한 실수 반올림 처리 이해.
 
#### 2026-07-01
- **학습 내용**: Java 기초 - 조건문과 반복문 (`_ch04`)
- **요약**:
    - **조건문 (if / switch)**:
      - if, if-else, if-else if 문을 사용한 조건 분기 처리 학습.
      - switch문을 통해 다중 분기를 간결하게 표현하는 방식과 break, fall-through 동작 원리 이해.
    - **반복문 (for / while / do-while)**:
      - for문의 초기화식·조건식·증감식 구조 및 중첩 for문 활용 실습.
      - while문과 do-while문의 차이 — 조건 검사 시점(선검사 vs. 후검사) 이해.
    - **분기문 (break / continue)**:
      - break를 이용한 반복 탈출 및 이름 붙은 반복문(labeled loop) 으로 중첩 루프 한 번에 탈출하는 방법 학습.
      - continue를 활용하여 특정 조건에서 나머지 반복 코드를 건너뛰는 흐름 제어 실습.
